PROJECTS



ML Skin & Color Analysis

Computer Vision, Machine Learning, Python, React

Program analyzes a picture of your face to parse skin type, acne concern level, complexion, hair color, undertones, and eye color to provide skin care, makeup, and fashion suggestions. Utilizes a KNN to classify skin tone on Fitzpatrick scale, and seasonal color for color analysis. In progress.



ML Nasal Disease Diagnosis Program

Machine Learning, Signal Processing Python

Machine learning diagnosis program for Chronic Rhinosinusitis, an expensive and painful nasal disease. Program processes speech recordings with many signal processing algorithms (TW, FFT, STFT, CWT) the results of which are fed into a Support Vector Machine (SVM) that diagnoses the patient. Cost is 1% of the most common diagnosis method (PROM-based clinician diagnosis) and accuracy is 12% higher. Paper in progress.
